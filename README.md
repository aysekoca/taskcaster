# TaskCaster - Task Management System

## Overview
TaskCaster is a full-stack task management application that allows users to create, manage, and track tasks. It includes authentication, role-based authorization, and a structured API.

## Technologies
Backend:
- Go (Fiber)
- JWT
- MySQL

Frontend:
- React
- Redux

## Features
- User authentication (login/register)
- Role-based access control
- Task management (create, update, delete)
- Admin panel and statistics
- File upload support

## Project Structure
- backend: API and server logic
- frontend: user interface

## Setup

Backend:
cd backend  
go mod tidy  
go run main.go  

Frontend:
cd frontend  
npm install  
npm start
